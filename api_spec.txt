1. Introduction
The Next Train API provides the train arrival schedule for MTR Airport Express Line, Tung Chung Line,
Tuen Ma Line, Tseung Kwan O Line, East Rail Line, South Island Line, Tsuen Wan Line, Island Line,
Kwun Tong Line and Disneyland Resort Line.

2. API
API Type: REST
HTTP Method: GET
Resource URL: https://rt.data.gov.hk/v1/transport/mtr/getSchedule.php
